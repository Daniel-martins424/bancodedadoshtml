Manual: Acessar o MySQL pela Linha de Comando no XAMPP

Passo 1: Abrir o Terminal (CMD)

Pressione Win + R, digite cmd e pressione Enter.

Passo 2: Navegar até o diretório do MySQL

Digite o seguinte comando (ajuste o caminho se necessário):

cd C:\xampp\mysql\bin

Passo 3: Acessar o MySQL

Execute o comando abaixo para acessar o MySQL como root:

mysql -u root

Se não houver senha definida para o usuário root (padrão do XAMPP), apenas pressione Enter.

Pronto!

Agora você está dentro do console do MySQL. Pode executar comandos SQL normalmente.

SHOW DATABASES;

Comandos SQL Essenciais

Comandos principais para gerenciar bancos de dados no MySQL:

-- Criar banco de dados
CREATE DATABASE nome_banco;

-- Usar banco de dados
USE nome_banco;

-- Criar tabela
CREATE TABLE nome_tabela (
    id INT PRIMARY KEY,
    nome VARCHAR(100)
);

-- Alterar tabela
ALTER TABLE nome_tabela ADD email VARCHAR(100);

-- Remover coluna
ALTER TABLE nome_tabela DROP COLUMN email;

-- Ver estrutura da tabela
DESCRIBE nome_tabela;

-- Inserir dados
INSERT INTO nome_tabela (id, nome) VALUES (1, 'João');

-- Atualizar dados
UPDATE nome_tabela SET nome = 'Maria' WHERE id = 1;

-- Deletar dados
DELETE FROM nome_tabela WHERE id = 1;

-- Condições (filtros)
SELECT * FROM nome_tabela WHERE nome = 'Maria';
SELECT * FROM nome_tabela WHERE idade > 18 AND cidade = 'SP';
SELECT * FROM nome_tabela WHERE NOT nome = 'João';

-- Buscar por padrão
SELECT * FROM nome_tabela WHERE nome LIKE 'Ma%';

-- Consultar dados
SELECT * FROM nome_tabela;

-- Junção entre tabelas
SELECT a.nome, b.salario
FROM funcionarios a
JOIN salarios b ON a.id = b.funcionario_id;
